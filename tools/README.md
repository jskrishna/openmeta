# Tools

Repo tooling for OpenMeta: scripts, generators, and release helpers.

| Folder | Purpose |
| ------ | ------- |
| [scripts/](./scripts/) | One-off and recurring maintenance scripts |
| [generators/](./generators/) | Scaffolding for packages, fields, docs, ADRs |
| [release/](./release/) | Versioning, changelog, and publish helpers |

Wire these into root `package.json` / Composer scripts as they land.
