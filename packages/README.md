# Packages

OpenMeta monorepo packages. Each folder will become a publishable unit (Composer and/or npm) as the design solidifies.

| Package | Role |
| ------- | ---- |
| [core/](./core/) | Field engine, models, WordPress plugin bootstrap |
| [blocks/](./blocks/) | Gutenberg block integrations |
| [graphql/](./graphql/) | GraphQL schema and resolvers |
| [react/](./react/) | Shared React UI for admin / builders |
| [cli/](./cli/) | Developer CLI |
| [sdk/](./sdk/) | Public SDK for extensions and integrations |
| [ai/](./ai/) | AI-assisted modeling and workflows |

> Pre-alpha: packages are stubs until architecture ADRs land.
