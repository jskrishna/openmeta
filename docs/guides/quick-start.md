# Quick Start

---

# Purpose

This guide introduces the fastest path to becoming productive with OpenMeta.

It demonstrates the overall workflow without exploring advanced configuration or customization.

---

# Goals

The Quick Start guide should:

- Introduce the framework
- Demonstrate the development workflow
- Build familiarity
- Reduce onboarding time
- Encourage exploration

---

- Learn one feature at a time.
- Explore documentation gradually.
