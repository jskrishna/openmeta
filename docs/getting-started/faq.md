# Frequently Asked Questions

---

# What is OpenMeta?

OpenMeta is a modern content modeling framework for WordPress that enables developers to create structured content using Schemas, Field Groups, Fields, Validation Rules, and Storage Drivers.

---

# Is OpenMeta a replacement for WordPress?

No.

OpenMeta extends WordPress while remaining fully compatible with its ecosystem.

---

# Is OpenMeta a replacement for ACF?

No.

OpenMeta serves a different purpose.

It focuses on framework architecture, extensibility, storage abstraction, and enterprise-scale application development rather than replacing existing custom field solutions.

---

# Can I use WordPress Meta?

Yes.

OpenMeta supports WordPress Meta as one of its Storage Drivers.

Applications may also use Custom Tables or other supported storage implementations.

---

# Does OpenMeta support Custom Tables?

Yes.

Custom Tables are supported through dedicated Storage Drivers.

Business logic remains independent of the underlying storage implementation.

---

# Can I build Custom Fields?

Yes.

Developers can create reusable Field Types through the extension system without modifying framework code.

---

# Does OpenMeta support REST APIs?

Yes.

OpenMeta provides integration with REST APIs for structured content access.

---

# Does OpenMeta support GraphQL?

Yes.

GraphQL integration is supported through dedicated modules and extensions.

---

# Can I create my own Storage Driver?

Yes.

Storage Drivers follow a common contract, allowing developers to implement alternative persistence mechanisms.

---

# Does OpenMeta support Multisite?

Yes.

OpenMeta is designed to operate within WordPress Multisite environments while maintaining proper site isolation.

---

# Can I extend the framework?

Yes.

OpenMeta is designed for extensibility through:

- Extensions
- Packages
- Service Providers
- Events
- Hooks
- Custom Fields
- Storage Drivers

---

# Is OpenMeta suitable for enterprise applications?

Yes.

OpenMeta is designed around clean architecture, dependency injection, modularity, and storage abstraction, making it suitable for large and long-lived applications.

---

# Is coding experience required?

Yes.

OpenMeta is intended for developers who are familiar with PHP and WordPress development.
