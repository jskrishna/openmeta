# Recipes

Short, focused how-tos for common tasks. For end-to-end walkthroughs see
[Tutorials](../tutorials/README.md); for copy-paste snippets see
[Examples](../examples/README.md).

- Scaffold a command: `php bin/openmeta make:command GreetCommand`
- Generate API docs: `php bin/openmeta docs:api`
- Validate documentation: `php bin/openmeta docs:validate`
- Boot the whole framework in one call: `OpenMeta\Framework\Framework::boot()`
